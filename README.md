
TOEICに頻出の英単語を４択問題を解くことで実戦形式で学べるアプリケーションです。
Webアプリケーション開発の第一歩として作成しました。


##主な機能
- **Practice**  TOEIC本番を想定した４択の英文穴埋め問題を制限時間以内に解くモードです。
                短文タイプと長文タイプのに２種類あり、実戦形式で学ぶことができます。
                問題のレベルが１～４まであり、制限時間を三段階で設定できます。

- **Review**    Practiceで間違った問題を復習するモードです。

- **Libarary**  問題で分からなかった単語を単語帳風のUIで確認することができます。
    


##使用技術
- **Frontend**: JavaScript (Vanilla JS), CSS, HTML
- **Backend**: Node.js, Express
- **Database**: PostgreSQL (Prisma ORM)
- **Authentication**: JWT (JSON Web Token)
